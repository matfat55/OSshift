# Product Context for OSshift

## Purpose

OSshift is a desktop application designed to provide users with an always-accessible AI assistant on Windows systems. It aims to make AI assistance seamlessly integrated into the user's workflow without requiring constant browser access or switching between applications.

## Problems Solved

- **Accessibility**: Provides instant access to AI assistance without disrupting workflow
- **Privacy**: Operates client-side with user-provided API keys, ensuring user data control
- **Efficiency**: Reduces context switching by being always available but unobtrusive
- **Customization**: Allows users to use their preferred AI models via their own API keys

## User Experience Goals

- **Minimal Footprint**: Small window that doesn't dominate screen real estate
- **Instant Access**: Quick toggle with right shift key to show/hide the interface
- **Seamless Integration**: Feels like a natural extension of the operating system
- **Responsive**: Quick responses to user queries without noticeable lag
- **Non-Intrusive**: Stays out of the way when not needed

## Target Users

- Knowledge workers who frequently need information or assistance
- Developers who want quick access to coding help
- Writers seeking real-time feedback or suggestions
- Anyone who regularly uses AI assistants and wants more convenient access

## Key Features

- Toggle visibility with right shift key
- Chat interface for AI interaction
- Client-side operation with user API keys
- Persistent across sessions
- Minimal resource usage
